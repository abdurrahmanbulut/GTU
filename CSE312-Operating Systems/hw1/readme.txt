To run the OS:
	1 -> Open terminal in makefile directory
	2 -> Write "make"
	3 -> It will create an .iso image file. this is your OS image.
	4 -> Install it to your machine with name "My Operating System"
	5 -> After installation, write "make" again to terminal
	6 -> Your OS will start to run.

Note: If you are using latest linux versions. If so, you will have VirtualBoxVM instead of VirtualBox. In this case, edit makefile and change "VirtualBox" with "VirtualBoxVM".
